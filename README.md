## Prerequisite


* [Angular 6] 
* [Bootstrap 4] 
* [Node.js ]


## Clone the Project

```bash 
git clone https://git.pccth.com/hardworkCloud/horus.git

```

## Install npm packages

Install the `npm` packages described in the `package.json` and verify that it works:

```bash
npm install

```
## Open Project
```bash
ng serve
```
